# Dashboard Xbox Game Pass Subscription Sales - Desafio de Projeto Santander Bootcamp Ciência de Dados 2025-2

O objetivo deste desafio é criar um dashboard de vendas, com foco na organização e visualização de dados, para transformar dados brutos em informações visuais claras e úteis, permitindo uma análise eficaz do desempenho de vendas e a tomada de decisões baseadas em dados. 

## Processo

Foi fornecida uma base de dados sobre as vendas de assinaturas do Game Pass da Xbox, com detalhamento de cada assinatura, como, por exemplo, qual o plano (Core, Standard ou Ultimate) e se incluía adicionais como o EA Play Season Pass e o Minecraft Season Pass. A base de dados foi analisada para entender seu conteúdo e, em seguida, encontrar padrões através de cálculos com tabelas dinâmicas, utilizando filtros para selecionar as informações requeridas.  

Para esse dashboard específico, foram utilizadas 5 perguntas de negócio para orientar os cálculos. São elas:

1 - Qual faturamento total de vendas de planos anuais (contendo todas as assinaturas agregadas)

2 - Qual faturamento total de vendas de planos anuais, separado por autorrenovação ou não

3 - Qual o total de vendas de assinaturas do EA Play Season Pass

4 - Qual o total de vendas de assinaturas do Minecraft Season Pass

5 - Qual o total de assinaturas de cada tipo (Core, Standard, Ultimate)

## Resultado

Os gráficos resultantes de tabelas dinâmicas criadas com base nos dados foram inseridos em uma nova planilha, chamada Dashboard, para unificar as visualizações de dados em um único local. Foram utilizados cores e logos dos itens mencionados para uma visualização mais limpa e agradável, de modo que as informações principais fiquem facilmente acessíveis.
